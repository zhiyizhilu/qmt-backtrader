# 多品种ETF动量轮动+EPO+风险平价混合策略

## 策略概述

基于年化收益和判定系数R²打分的动量因子轮动策略，结合EPO（Enhanced Portfolio Optimization）与风险平价混合权重优化。ETF池覆盖商品、海外、宽基和窄基，构成完整美林时钟，通过动量轮动捕捉趋势性机会，混合权重兼顾信号驱动与风险分散。

原始策略来源: 聚宽社区 openhe https://www.joinquant.com/post/47208

## 交易逻辑

1. 每月首个交易日调仓
2. 动量打分：对ETF池中每只ETF，用近34日收盘价做对数线性回归，年化收益率 × R² 作为动量分数
3. 过滤掉动量分数<=0的ETF，取前3只
4. 权重计算：60% EPO优化权重 + 40% 风险平价权重
   - EPO：用anchored方法计算持仓权重（锚定为逆方差权重），收缩相关矩阵降低估计误差
   - 风险平价：逆波动率等风险贡献权重，降低集中风险
5. 按混合权重分配资金买入

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| momentum_days | 34 | 动量参考天数（对数线性回归窗口） |
| top_n | 3 | 持有动量排名前N只ETF |
| epo_lambda | 10 | EPO风险厌恶系数，越高越保守 |
| epo_w | 0.2 | EPO模糊化参数/收缩权重，越高权重越分散 |
| epo_lookback | 1200 | EPO优化回看天数（约5年） |
| risk_parity_blend | 0.4 | EPO与风险平价混合比例，0=纯EPO，1=纯风险平价 |
