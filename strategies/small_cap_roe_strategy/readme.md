# 小市值ROE/ROA策略

## 策略概述

克隆自聚宽文章: https://www.joinquant.com/post/45510

核心思路：选择ROE>15%且ROA>10%的小市值股票，月度调仓。巴菲特说如果只用一个指标选股，他会留下ROE（净资产收益率）；而ROA（总资产回报率）则衡量公司对总资产的赚钱能力，排除高杠杆带来的虚高ROE。

## 选股逻辑

1. 过滤科创板（688xxx）、北交所（4/8开头）股票
2. 过滤ST股票
3. 过滤停牌股票
4. 过滤次新股（上市不足250天）
5. 过滤涨停/跌停股票（已持仓除外）
6. 过滤高价股（股价>=10元，已持仓除外）
7. 基本面筛选：ROE > 15%, ROA > 10%
8. 按市值升序排列（小市值优先）
9. 剔除黑名单（30天内持有过且30天内涨停过的股票）
10. 取前10只股票

## 调仓规则

- 月度调仓，等权重持仓
- 最多持仓10只股票
- 每日检查持仓中昨日涨停的股票，若打开涨停则卖出
- 涨停卖出后若持仓不足10只，补仓选股

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| rebalance_freq | monthly | 调仓频率 |
| max_stocks | 10 | 最大持仓数量 |
| position_ratio | 0.95 | 仓位比例 |
| min_roe | 0.15 | 最低ROE阈值 |
| min_roa | 0.10 | 最低ROA阈值 |
| max_price | 10.0 | 最高股价（元） |
| limit_days | 30 | 黑名单回看天数 |
| min_ipo_days | 250 | 最低上市天数 |
