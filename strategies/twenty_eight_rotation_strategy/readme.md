# 二八轮动小市值策略

## 策略概述

基于沪深300和中证500指数20日涨幅进行市场择时，在市场趋势向上时选取小市值股票持仓，在市场趋势向下时清仓避险。源自聚宽社区经典策略"二八轮动小市值优化版"。

## 选股逻辑

1. 每日检查沪深300ETF(510300.SH)和中证500ETF(510500.SH)的20日涨幅
2. 如果两个指数20日涨幅都为负，清仓避险（二八止损）
3. 如果至少一个指数20日涨幅为正，按调仓频率进行小市值选股调仓
4. 选股：过滤停牌/涨停/跌停/创业板股票，按市值升序排列，选取前N只

## 止损机制

### 二八止损（必需）
- 每日检查沪深300和中证500指数的20日涨幅
- 如果两个指数涨幅都为负，则清仓，重置调仓计数，待下次调仓条件满足再操作

### 大盘止损（可选）
1. **价格止损**：检查大盘(沪深300指数)前130日的最低价和最高价，如果最高价大于最低价的2倍则清仓，停止交易
2. **三只黑鸦止损**：判断大盘是否呈现三只黑鸦形态（连续3日阴线且收盘价逐步走低），如果是则当天清仓并停止交易，第二天停止交易一天

### 个股止损（可选）
- 每日判断个股是否从持仓后的最高价回撤，如果超过个股回撤阈值（默认15%），则平掉该股持仓

## 调仓规则

- 每5个交易日调仓，等权重持仓
- 最多持仓 3 只股票
- 仓位比例 95%
- 择时信号为负时清仓

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| rebalance_interval | 5 | 调仓间隔（交易日） |
| max_stocks | 3 | 最大持仓数量 |
| position_ratio | 0.95 | 仓位比例 |
| momentum_period | 20 | 择时动量周期（日） |
| filter_chinext | True | 是否过滤创业板 |
| filter_limit_down | True | 是否过滤跌停股 |
| max_market_cap | None | 市值上限（亿元） |
| enable_market_stoploss | True | 是否启用大盘价格止损 |
| market_price_period | 130 | 大盘价格止损周期（日） |
| market_price_ratio | 2.0 | 大盘价格止损高低价比值阈值 |
| enable_three_crows | True | 是否启用三只黑鸦止损 |
| three_crows_period | 4 | 三只黑鸦判断周期（日） |
| enable_stock_stoploss | True | 是否启用个股止损 |
| stock_stoploss_threshold | 0.15 | 个股止损回撤阈值 |

## 回测配置

- 初始资金: 1,000,000
- 股票池: **中证全指**
- 回测区间: 2020-04-28 ~ 2026-04-28
- 基准指数: 000852.SH
- 佣金: 0.0003
- 数据源: QMT（含财务数据缓存）

## 快速开始

```bash
python main.py --mode backtest --strategy twenty_eight_rotation --period 1d --pool 中证全指 --start 2020-04-28 --end 2026-04-28 --ai-mode
```

## 数据来源

- 行情数据: QMT（国金证券 xtquant）
- 财务数据: QMT（资产负债表、利润表、现金流量表等）
- 指数ETF数据: 510300.SH（沪深300ETF）、510500.SH（中证500ETF）
- 大盘基准: 000300.SH（沪深300指数，用于大盘止损判断）
- 股票池: **中证全指**（全市场A股历史成分股）

## 注意事项

- 原始策略使用3日调仓间隔，本实现默认5日，可通过 `rebalance_interval` 参数调整
- 原始策略使用分钟级回测，本实现使用日线级别回测
- 市值计算使用总股本×当前价格，当财务数据不可用时回退为价格排序
- 大盘止损和个股止损均可通过参数独立开关控制
