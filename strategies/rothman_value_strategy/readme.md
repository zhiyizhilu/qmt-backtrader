# 霍华·罗斯曼审慎致富价值精选策略

## 策略概述

源自华尔街远见投资顾问公司(VIA)首席分析师Howard Rothman的投资哲学："在适当价格买入稳定且持续成长获利的公司股票"。该策略偏好大型股，强调公司产生现金的能力和稳定成长能力，通过6大财务条件筛选优质价值蓝筹股，旨在穿越牛熊、基业长青。

## 选股逻辑

1. 总市值 >= 200亿（偏好大型股，对应原策略50亿美元标准）
2. 流动比率 >= 1.0（良好的财务结构，短期偿债能力；金融股数据缺失时跳过）
3. ROE >= 8%（较高的股东权益报酬率）
4. 每股经营现金流 > 0（拥有良好且持续的现金流）
5. 净利润同比增长率 >= 0%（稳定持续的盈利成长）
6. 盈余报酬率(EPS/Price) >= 2%（优于市场平均的盈余报酬率；数据缺失时跳过）

**综合评分公式**：ROE×0.35 + max(利润增速,0)×0.25 + 盈余报酬率×0.25 + (OCFPS>0)×0.15

## 调仓规则

- 月度调仓，等权重持仓
- 最多持仓 20 只股票
- 仓位比例 95%

```bash
python main.py --mode backtest --strategy rothman_value --period 1d --pool 沪深300 --start 2020-04-28 --end 2026-04-28 --ai-mode
```

## 数据来源

- 市值: 总股本(净资产/每股净资产) × 未复权价格
- 流动比率: Balance.total_current_assets / Balance.total_current_liability
- ROE: Pershareindex.du_return_on_equity
- 每股经营现金流: Pershareindex.s_fa_ocfps
- 净利润增速: Pershareindex.inc_net_profit_rate
- 盈余报酬率: Pershareindex.eps_diluted / 未复权价格

## 策略优化建议

当前回测结果表现不佳，建议使用 `strategy-optimizer` 技能进行系统性优化，可调方向包括：
- 提高ROE门槛（如0.10或0.12）
- 提高利润增速门槛（如0.05或0.10）
- 调整评分权重
- 缩小持仓数量（如10只或15只）
- 更换股票池（如中证500或中证1000）
