# 中小综指小市值策略 (jq\_small\_cap)

## 策略逻辑

从中小板股票池中，按流通市值升序选股，每日调仓，等权持仓。

### 选股流程

1. 获取股票池（中小综指成分股）
2. 过滤停牌、涨停、跌停股票
3. 计算每只股票的流通市值（总股本 × 收盘价）
4. 按市值升序排列，选取前 N 只
5. 等权重持仓，仓位比例 95%

### 策略参数

| 参数               | 默认值     | 说明                    |
| ---------------- | ------- | --------------------- |
| `rebalance_freq` | `daily` | 调仓频率                  |
| `max_stocks`     | `5`     | 持仓股票数量                |
| `position_ratio` | `0.95`  | 仓位比例                  |
| `stock_pool`     | `None`  | 自定义股票池（None 则使用注册时的池） |
| `max_market_cap` | `None`  | 市值上限（亿元），None 表示不限制   |

### 市值计算方式

```
总股本 = 总权益 / 每股净资产
流通市值 = 总股本 × 未复权收盘价
```

若每股净资产数据缺失，则直接使用总权益作为市值近似。

## 风险提示

- 该策略每日调仓，换手率极高，实盘交易成本可能显著高于回测假设
- 小市值股票流动性较差，实盘滑点可能较大
- 回测结果不代表未来表现
