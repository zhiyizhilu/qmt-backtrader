# 首板低开策略

## 策略概述

首板低开策略选取"低位、非连板、涨停后次日低开"的股票，开盘买入，第二日上午如果有盈利就卖出，没有就拿到尾盘无论盈利与否都卖出。

来源: 聚宽社区 https://www.joinquant.com/view/community/detail/44901
作者: wywy1995

本策略提供两个版本：
- **日线版** (`first_board_low_open`)：每日调仓，框架级回测
- **分钟线版** (`first_board_low_open_1m`)：精确模拟日内交易时点，更贴近实盘

## 选股逻辑

1. 筛选昨日涨停的股票（收盘价等于涨停价）
2. 排除连板股票（前日也涨停的为连板，排除）
3. 计算相对位置，只保留60日内相对位置<=50%的低位股
4. 筛选今日低开3%-4%的股票（开盘价/昨收在0.96-0.97之间）

## 交易规则

### 日线版

- 每日调仓，等权重持仓
- 买入价格为当日收盘价（非开盘价），与原始策略有差异
- 持仓股次日若未涨停则卖出（通过日度调仓自动实现）

### 分钟线版

分钟线版本精确模拟了原始策略的日内交易逻辑：

| 时间 | 操作 |
|------|------|
| 09:31 | 开盘买入选中的股票 |
| 11:28 | 检查持仓，有盈利则卖出（止盈） |
| 14:50 | 尾盘卖出所有剩余持仓（无论盈亏） |

**T+1规则**：当日买入的股票不可当日卖出，所有卖出操作在T+1日执行。

## 分钟线数据工程

分钟线回测涉及大量数据工程问题，以下为关键处理：

### 数据源拼接

QMT分钟线API仅提供最近约1年的数据，回测6年需要多源拼接：
- **QMT数据优先**：覆盖最近1年，数据最新最准
- **聚宽数据补足**：覆盖2020年至今的历史区间，填补QMT缺失部分
- 合并策略：QMT区间用QMT数据，QMT无数据的区间用聚宽转换数据

### 后复权转换

聚宽仅提供不复权分钟数据，需转换为后复权。核心发现：**QMT后复权与不复权是严格线性关系 `hfq = bfq × a + b`**，利用QMT日线数据拟合每天的a、b参数，对聚宽不复权数据做线性变换。验证结果：close精确匹配率99.95%。

### QMT后复权负价格问题

QMT后复权算法从当前往回推算，多次现金分红使参数b变负，当 `bfq × a < |b|` 时产生负价格。37只策略股票中4只受影响（600734.SH、600076.SH、600052.SH、600744.SH），其中前2只较严重，已在回测中做排除处理。

### 数据验证

每次数据转换后自动执行5项质量检查：重叠区间对比、边界连续性、价格合理性、负价格检测、数据完整性。详见 `convert_minute_hfq.py` 中的 `validate_merged_data` 函数。


## 概率统计回测

除框架级回测外，本策略还提供独立的概率统计回测脚本，用于逐只逐次模拟交易并生成详细统计报告。

### 分钟级别统计回测

```bash
python minute_statistical.py
```

**功能**：基于分钟线数据，模拟 T+1 交易规则
- 买入：T 日 09:31 开盘价
- 卖出：T+1 日 11:28 若有盈利则卖出，否则 14:50 尾盘清仓

**输出**：`分钟回测结果.csv`

**统计指标**：胜率、平均盈利/亏损、盈亏比、按年份分布、盈利分布等

### 日线级别统计回测

```bash
python daily_statistical.py
```

**功能**：基于日线数据，模拟 T+1 交易规则
- 买入：T 日开盘价
- 卖出：T+1 日开盘价若盈利则卖出，否则 T+1 日收盘价卖出

**输出**：`日回测结果.csv`，并自动与分钟线结果对比

**对比指标**：
| 指标 | 日线统计 | 分钟线统计 |
|------|---------|-----------|
| 胜率 | 72.41% | 62.56% |
| 盈亏比 | 1.05 | 1.39 |
| 平均单次收益 | 1.73% | 1.80% |

### 数据准备

运行统计回测前，需确保 `screened_stocks_raw.txt` 存在，格式如下：
```
000007.SZ: 1 次, 日期: 2021-04-01
000010.SZ: 2 次, 日期: 2021-04-01, 2022-05-12
...
```

该文件可通过选股脚本生成，或从现有 CSV 回测结果中提取。

## 数据来源

- 行情数据: QMT数据源（优先）+ 聚宽数据（补充历史）
- 涨停价计算: 基于前收盘价 × (1 + 涨停比例)，主板10%，创业板/科创板20%，北交所30%
- 创业板20%涨跌幅自2020-08-24起生效，此前为10%
- 后复权转换: `convert_minute_hfq.py`，基于QMT日线参数的线性变换

## 已知问题

1. **日线版买入价偏差**：日线回测买入价为收盘价而非开盘价，与策略意图不符，建议使用分钟线版
2. **QMT后复权负价格**：QMT算法在极端情况下（股价极低+分红极多）产生负价格，属数据源缺陷
3. **成交量口径差异**：QMT与聚宽的成交量统计口径不同（偏差~99%），当前策略不依赖成交量
4. **涨停判断精度**：使用 `close >= high_limit - 0.005` 容差，与聚宽精确字段可能有细微差异
