# 高股息行业均仓策略

## 一、策略概述

高股息策略的核心逻辑是：选择现金流健康、分红可持续、净利润稳定增长的公司，通过行业分散配置，构建一个低波动、稳健收益的量化组合。

与小市值策略相比，高股息策略波动更小，更符合价值投资理念，适合作为策略池的"压舱石"。

## 二、核心风险：高股息陷阱

### 1. 分红不可持续
很多公司看重股息率高，实际上基本面逐渐恶化、债务高企，强行分红吸引散户；或者某一年大额分红，第二年亏损。

### 2. 行业集中度风险
高股息股票往往集中在特定行业（钢铁、煤炭、房地产等），行业景气高点时分红丰厚，但行业到顶后股价可能大幅下跌。此时股息率 = 每股分红 ÷ 股价，可能股息率未变，但股价已跌得惨不忍睹。

## 三、选股流程

```
股票池 → 基本面三重过滤 → 计算股息率 → 行业分散选股 → 优中选优 → 取前N只
```

### 1. 基本面三重过滤

| 条件 | 数据来源 | 字段 | 说明 |
|------|---------|------|------|
| ROE > 0 | Pershareindex | `du_return_on_equity` | 确保公司盈利，非亏损吃老本分红 |
| 归母净利润增速 > 0 | Pershareindex | `inc_net_profit_rate` | 确保业绩增长，分红可持续 |
| 经营现金流 > 0 | Pershareindex | `s_fa_ocfps` | 确保分红来自真金白银，非数字游戏 |

三个条件同时满足，选出的公司既有盈利能力、又有成长性、还有健康的现金流——这才具备持续分红的基础。

### 2. 股息率计算

```
股息率 = 每股分红 / 当前股价
```

支持 `use_avg_dividend` 模式：取近3年股息率均值，提升稳定性，避免单一年度异常分红干扰。

### 3. 行业分散选股

- 申万31个一级行业各自拆开
- 每个行业挑选股息率最高的1只股票
- 再从这31只中优中选优、取前N名

这样选出来的就是高分红行业中的TOP股票，同时天然规避单一行业集中暴露的风险。

## 四、策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `rebalance_freq` | `'biweekly'` | 调仓频率（双周调仓） |
| `max_stocks` | `15` | 最大持仓股票数 |
| `position_ratio` | `0.95` | 仓位比例，5%留作现金缓冲 |
| `stock_pool` | `None` | 股票池，None则从财务数据缓存获取 |
| `min_roe` | `0.0` | ROE最低阈值 |
| `min_profit_growth` | `0.0` | 归母净利润增速最低阈值 |
| `min_operate_cashflow` | `0.0` | 经营现金流最低阈值 |
| `use_avg_dividend` | `True` | 使用近3年平均股息率 |
| `skip_fundamental_if_missing` | `True` | 所有股票财务数据缺失时跳过基本面过滤 |

## 七、风险提示

1. **分红不可持续**：高股息是结果，不是原因。需持续跟踪公司基本面变化
2. **行业周期风险**：即使行业分散，系统性风险仍无法完全规避
3. **利率环境变化**：利率上升时，高股息策略吸引力下降
4. **历史回测不代表未来**：任何历史表现都不能保证未来收益
