# ARBR因子+小市值选股策略

## 策略概述

基于ARBR情绪因子筛选+小市值排序的选股策略。克隆自聚宽文章"10年52倍，年化59%，全新因子方法超稳定"。

核心思路：通过ARBR情绪因子（人气意愿指标）筛选出情绪适中的股票（标准化值在[-1, 1]范围内），再按市值升序排列，选取最小市值的3只股票等权持仓。

## 选股逻辑

1. 从中证1000成分股中过滤：剔除创业板/科创板/北交所/ST/停牌/退市/涨跌停/次新股
2. 计算ARBR因子值（AR-BR差值），MAD去极值后标准化，筛选z-score在[-1, 1]范围内的股票
3. 按总市值升序排列
4. 过滤EPS<=0的股票
5. 取前3只

## 调仓规则

- 每周调仓，等权重持仓
- 最多持仓 3 只股票
- 仓位比例 95%
- 4月5日-4月30日空仓
- 持仓中昨日涨停的股票，若今日打开涨停则卖出
- 个股止损线9%，止盈线100%（成本价2倍）

## 已知偏差（vs聚宽原版）

- 原版使用聚宽jqfactor的ARBR因子，本版自行计算ARBR指标
- 原版使用valuation.circulating_market_cap（流通市值），本版用cap_stk*price（总市值）近似
- 原版使用indicator.eps，本版用Pershareindex.s_fa_eps_diluted
- 原版因子预处理含行业均值填充NaN，本版简化为直接剔除NaN
- 原版涨停检查使用1分钟数据，本版使用日线数据近似判断
