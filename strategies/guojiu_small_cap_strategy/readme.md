# 国九小市值策略 (Guojiu Small Cap Strategy)

> 策略注册名：`guojiu_small_cap`
> 策略类型：选股调仓型 (StockSelectionStrategy)
> 克隆自聚宽：https://www.joinquant.com/view/community/detail/47791

## 策略概述

基于"国九条"财务筛选标准的小市值选股策略。从中小企业板指数(399101.SZ)成分股中，通过国九条财务指标（净利润>0、归母净利润>0、营业收入>1亿）过滤后，按总市值升序选取最小市值股票，等权重持仓并每周调仓。策略在1月和4月空仓规避风险，同时设有个股止损、止盈和市场大跌止损机制。

## 选股逻辑

1. 从**中小企业板指数(399101.SZ)**成分股中筛选，过滤ST/退市/科创北交/创业板/次新股/停牌/涨跌停
2. **国九条过滤**：净利润>0、归母净利润>0、营业收入>1亿
3. 按总市值升序排列，取**前N只**最小市值股票（候选池为 stock_num×3）
4. 过滤股价超过上限的股票（已持有的不受限制）

## 调仓规则

- **每周调仓**，等权重持仓
- **动态持股数量**：根据中小企业板指数MA10与收盘价差值调整（5~10只）
  - 差值 >= 500：5只（市场强势，集中持仓）
  - 200 <= 差值 < 500：6只
  - -200 <= 差值 < 200：8只（震荡）
  - -500 <= 差值 < -200：9只
  - 差值 < -500：10只（市场弱势，分散持仓）
- **1月和4月空仓**（持有现金）
- 持仓中昨日涨停的股票，若今日打开涨停则卖出
- 涨停卖出后次日补仓（从目标列表中选新股票买入）
- **个股止损9%**，**盈利100%止盈**
- **市场大跌止损**：中小企业板指数成分股平均跌幅>=5%时清仓

### 差异分析

本地实现与聚宽原版存在较大差异，主要原因：

1. **市值计算**：聚宽使用内置`valuation.market_cap`，本版用`cap_stk*price`（财报总股本×股价），存在时点差异
2. **ETF现金管理**：原版空仓月份持有银华日利ETF(511880)，本版空仓月份持有现金
3. **涨停检查**：原版使用1分钟数据在14:00检查，本版使用日线数据近似判断
4. **数据源差异**：QMT与聚宽的财务数据、行情数据存在细微差异

## 已知偏差（vs聚宽原版）

- 原版空仓月份持有银华日利ETF(511880)，本版空仓月份直接清仓持有现金
- 原版审计意见过滤默认关闭，本版同样不实现
- 原版涨停检查使用1分钟数据，本版使用日线数据近似判断
- 原版市值使用valuation.market_cap（聚宽内置），本版用cap_stk*price（总股本×股价）
- 原版佣金结构与本版一致（买入万2.5/卖出万2.5+千1印花税/最低5元）
