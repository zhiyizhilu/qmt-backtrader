# JQData 数据下载说明

本目录下的脚本用于在 [聚宽(JoinQuant)](https://www.joinquant.com/) 平台上生成历史数据，下载后保存到本地使用。

**下载步骤：**

1. 上传 Notebook 到聚宽研究环境：
   - `获取指数成分股.ipynb` — 生成指数历史成分股数据
   - `获取行业成分股.ipynb` — 生成申万一级行业历史成分股数据

2. 在聚宽运行 Notebook，生成 CSV 文件

3. 下载 CSV 文件，链接格式：
   ```
   https://www.joinquant.com/user/{user_id}/files/{filename}.csv?download=1
   ```

4. 保存到 `.cache/JQData/` 对应目录

**CSV 文件格式：**

```csv
date,codes
2016-04-28,"['000001.SZ', '000002.SZ', ...]"
2016-06-13,"['000001.SZ', '000008.SZ', ...]"
```

### 2. 股票分钟线数据

用于下载单只股票的不复权分钟线行情数据。

**脚本文件：**
- `下载股票历史分钟线数据.ipynb` — 下载指定股票的分钟线数据

**使用方法：**

1. 将 `下载股票历史分钟线数据.ipynb` 上传到聚宽研究环境

2. 修改脚本顶部的 `CONFIG` 配置参数：
   ```python
   SECURITY = '000001.XSHE'      # 股票代码（聚宽格式）
   START_DATE = '2024-01-01'     # 开始日期
   END_DATE = '2024-12-31'       # 结束日期
   FREQUENCY = '1m'              # 分钟线频率: '1m', '5m', '15m', '30m', '60m'
   ```

3. 运行脚本，自动下载并保存为 CSV 文件

4. 点击输出的下载链接获取 CSV 文件

**CSV 文件格式：**

```csv
code,datetime,open,close,high,low,volume,money
000001.SZ,2024-01-02 09:31:00,10.50,10.52,10.55,10.48,12500,131250.0
000001.SZ,2024-01-02 09:32:00,10.52,10.51,10.53,10.50,8900,93450.0
...
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| `code` | 股票代码（通用格式：000001.SZ / 600000.SH） |
| `datetime` | 时间戳（格式：YYYY-MM-DD HH:MM:SS） |
| `open` | 开盘价 |
| `close` | 收盘价 |
| `high` | 最高价 |
| `low` | 最低价 |
| `volume` | 成交量（股） |
| `money` | 成交额（元） |

**注意事项：**
- 数据为**不复权**数据
- 分钟线频率支持 1分钟、5分钟、15分钟、30分钟、60分钟
- 聚宽对分钟线数据有下载限制，建议分批次下载较长时间段的数据
