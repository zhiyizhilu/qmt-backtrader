## 下载步骤

### 1. 上传 Notebook 到聚宽

将以下两个 Notebook 上传到聚宽研究环境：

- `获取指数成分股.ipynb` — 生成指数历史成分股数据
- `获取行业成分股.ipynb` — 生成申万一级行业历史成分股数据

### 2. 在聚宽运行 Notebook

依次运行 Notebook 中的代码单元格，Notebook 会调用聚宽 API 获取历史成分股数据并生成 CSV 文件。

### 3. 下载 CSV 文件

运行完成后，Notebook 会输出聚宽文件下载链接，格式如：

```
https://www.joinquant.com/user/{user_id}/files/{filename}.csv?download=1
```

点击链接即可下载对应的 CSV 文件。

### 4. 保存到本地目录

将下载的 CSV 文件按以下规则放入 `.cache/JQData/` 目录：

| 数据类型 | 文件示例 | 保存位置 |
|---------|---------|---------|
| 指数成分股 | `000300.SH.csv` | `.cache/JQData/index_constituent/` |
| 行业成分股 | `SW1银行.csv` | `.cache/JQData/industry_constituent/` |

## CSV 文件格式

两种数据的 CSV 格式一致：

```csv
date,codes
2016-04-28,"['000001.SZ', '000002.SZ', ...]"
2016-06-13,"['000001.SZ', '000008.SZ', ...]"
```

- **date**：成分股发生变更的日期
- **codes**：变更后的成分股代码列表（Python 列表格式的字符串）

每行代表一次成分股变更，两个日期之间的成分股保持不变。查询时取小于等于查询日期的最近一行。
