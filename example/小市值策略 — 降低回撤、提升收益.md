# 小市值策略优化思路 — 降低回撤、提升收益

## 一、小市值策略的本质与困境

小市值策略的核心逻辑是：小盘股由于机构覆盖少、信息不对称大、流动性溢价低，长期存在超额收益（Size Factor）。A股历史上小市值因子表现极为强势，年化收益远超大盘。

但问题同样突出：

- **回撤大**：小盘股在熊市中跌幅远超大盘，2015年股灾、2018年去杠杆、2022年成长股杀估值，小市值策略回撤动辄40%~60%
- **收益不稳定**：小盘股风格切换剧烈，牛市暴涨、熊市暴跌，收益分布严重右偏
- **流动性风险**：小盘股成交低迷，冲击成本高，实盘滑点大
- **壳价值衰减**：注册制推行后，壳资源价值大幅缩水，小市值因子长期溢价在收窄

## 二、降低回撤的核心思路

### 1. 基本面质量过滤 — 剔除"垃圾小盘"

小市值策略最大的坑，是选到了基本面恶化的"垃圾股"。纯按市值排序选股，容易选到亏损、高负债、现金流断裂的公司，这些股票在熊市中跌幅最深。

**过滤条件：**
- **ROE > 0**：至少是盈利公司，剔除亏损股
- **营收增速 > 0**：业务在增长而非萎缩
- **经营现金流 > 0**：赚的是真金白银
- **资产负债率 < 60%**：避免高杠杆公司（可选）

> 这与高股息策略的基本面过滤逻辑一致，但目的不同：高股息是为了分红可持续，小市值是为了排除"地雷"。

### 2. 行业分散 — 避免集中暴露

小盘股往往集中在特定行业（如TMT、机械、化工），行业集中会导致：
- 行业周期下行时，组合整体大幅回撤
- 某些行业的小盘股质量普遍较差

**做法：**
- 申万一级行业分类，每个行业最多选1~2只
- 剔除小盘股过于集中的行业（如房地产、钢铁）
- 行业中性化处理：每个行业选市值最小的，而非全市场选市值最小的

### 3. 动量+反转结合 — 避免接飞刀

纯小市值策略容易在下跌趋势中不断买入"便宜"的股票，结果越买越跌。

**改进：**
- **短期动量过滤**：过去20日涨幅 > 0，只买处于上升通道的小盘股
- **中期反转加成**：过去60日跌幅排名靠前的给予额外权重（超跌反弹逻辑）
- 两者结合：短期动量为正 + 中期超跌 = 最佳买点

### 4. 波动率调仓 — 高波动时降仓

小盘股的波动率在市场恐慌时会急剧上升，这是回撤的主要来源。

**做法：**
- 计算持仓组合的近20日波动率
- 波动率超过阈值时，自动降低仓位（如从95%降至60%）
- 波动率回落时，恢复满仓
- 类似于"风险平价"思路：波动大时少配，波动小时多配

### 5. 止损与回撤控制 — 主动风控

框架已内置 `RiskController`，小市值策略必须启用：

```python
from core.strategy_logic import RiskController

risk = RiskController(
    max_position_ratio=0.9,       # 总仓位上限90%（留10%现金缓冲）
    max_drawdown_limit=0.15,       # 最大回撤15%时停止买入
    stop_loss_ratio=0.08,          # 单股止损8%
    max_single_order_ratio=0.15,   # 单只股票不超过15%
)
strategy.set_risk_controller(risk)
```

**关键参数说明：**
- `stop_loss_ratio=0.08`：小盘股波动大，止损不宜过紧（5%容易被震出局），8%是平衡点
- `max_drawdown_limit=0.15`：组合回撤超15%时停止建仓，避免在暴跌中继续加仓
- `max_single_order_ratio=0.15`：单只不超过15%，10只股票等权配置

### 6. 流动性过滤 — 避免实盘无法成交

回测中容易忽略流动性，但实盘中这是致命问题。

**过滤条件：**
- 日均成交额 > 1000万（或日均换手率 > 1%）
- 排除ST、*ST、停牌股
- 排除上市不满60个交易日的次新股（波动过大、筹码不稳）

## 三、提升收益的核心思路

### 1. 多因子叠加 — 小市值不是唯一因子

纯小市值因子在A股近年表现衰减，需要叠加其他强势因子：

| 叠加因子 | 逻辑 | 实现方式 |
|---------|------|---------|
| **盈利质量** | 小盘+盈利 = 被忽视的好公司 | ROE > 阈值排序 |
| **成长性** | 小盘+高增长 = 成长股溢价 | 营收/利润增速排序 |
| **低估值** | 小盘+低估值 = 双重安全边际 | PE/PB分位数筛选 |
| **技术动量** | 小盘+动量 = 趋势确认 | 过去N日涨幅排序 |

**推荐组合：小市值 + 高ROE + 营收增长**，这三种因子在A股的IC值长期为正且相关性低。

### 2. 调仓频率优化

- **月度调仓**：小市值策略的标准频率，平衡换手率和因子时效
- **避免周度调仓**：换手率过高，交易成本侵蚀收益
- **季度调仓**：如果叠加基本面因子，季度调仓更合理（财报季度更新）

### 3. 择时对冲 — 大盘弱势时减仓

小盘股的Beta远高于1，系统性风险是回撤的主要来源。

**简单择时：**
- 沪深300均线趋势：20日均线 > 60日均线时满仓，反之半仓
- 市场宽度：上涨个股占比 < 30%时降仓
- VIX类指标：波动率飙升时降仓

> 注意：择时是一把双刃剑，过度择时会错过反弹。建议只做"防守型择时"——判断市场极端时减仓，而非频繁择时。

### 4. 打新增强 — 小市值的天然优势

小市值策略持仓分散、市值配置合理，天然适合打新：
- 底仓满足打新门槛
- 打新收益作为增强，年化可增厚2%~5%

## 四、策略框架实现要点

基于本项目 `StockSelectionStrategy` 基类，小市值策略的核心实现：

```python
from core.stock_selection import StockSelectionStrategy
from core.strategy_logic import RiskController
from strategies import register_strategy


@register_strategy('small_cap', default_kwargs={'max_stocks': 10},
                   backtest_config={'cash': 1000000, 'commission': 0.0001,
                                    'start_date': '2016-01-01', 'end_date': '2026-04-17'})
class SmallCapStrategy(StockSelectionStrategy):
    """小市值优化策略 - 基本面过滤+行业分散+动量确认

    选股逻辑：
    1. 流动性过滤：日均成交额 > 阈值
    2. 基本面过滤：ROE > 0、营收增速 > 0、经营现金流 > 0
    3. 行业分散：申万一级行业各选市值最小的1只
    4. 动量确认：近20日涨幅 > 0
    5. 按市值升序排列，取前N只
    6. 等权重持仓，月度调仓
    """

    params = (
        ('rebalance_freq', 'monthly'),
        ('max_stocks', 10),
        ('position_ratio', 0.95),
        ('stock_pool', None),
        ('min_roe', 0.0),
        ('min_revenue_growth', 0.0),
        ('min_operate_cashflow', 0.0),
        ('min_avg_amount', 1e7),        # 日均成交额最低1000万
        ('momentum_period', 20),          # 动量回看期
    )
```

## 五、预期效果与风险提示

| 指标 | 纯小市值 | 优化后小市值 |
|------|---------|------------|
| 年化收益 | 25%~35% | 15%~25% |
| 最大回撤 | 40%~60% | 20%~30% |
| 夏普比率 | 0.8~1.0 | 1.0~1.5 |
| 卡玛比率 | 0.5~0.8 | 0.8~1.2 |
| 月度胜率 | 55%~60% | 60%~65% |

**核心取舍：** 优化后的策略牺牲了部分牛市中的爆发力，但大幅降低了熊市中的回撤，长期复利效果更优。

**风险提示：**
1. 注册制全面推行后，小市值因子长期溢价可能持续收窄
2. 小盘股流动性差，实盘冲击成本远高于回测
3. 任何历史回测都不能保证未来收益
4. 风控参数需要根据个人风险承受能力调整

## 六、总结

小市值策略的优化，本质上是在**收益和回撤之间寻找最优解**：

1. **基本面过滤**是最重要的改进——排除垃圾股，回撤立减1/3
2. **行业分散**是第二重要的改进——避免行业集中风险
3. **风控止损**是保命线——回撤超限时果断降仓
4. **动量确认**是加分项——避免在下跌趋势中接飞刀
5. **波动率调仓**是进阶优化——自适应市场环境

记住：**活下来比赚得多更重要**。小市值策略最大的敌人不是选不到好股票，而是在熊市中扛不住回撤而割肉。控制住回撤，收益自然会来。
